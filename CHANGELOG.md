# Changelog ‚Äî Hearthbound Hollow

All notable changes to this project will be documented here. Entries follow the [Keep a Changelog](https://keepachangelog.com/en/1.1.0/) format.

## [0.1.1-mission-1-2-bugfix-and-tooling] ‚Äî 2026-05-24

**Branch:** `feat/mission-1-2-architecture` ‚Üí PR #7 (open)
**Theme:** Bug-fix cycle (Phase 10.5) + quality-of-life tooling (Phase 11).

### Fixed
- **CS1739 in SaveService.cs** ‚Äî `VillageStateSavedEvent` ctor parameter is named `autosave`, not `isAutosave`. Switched the call site to positional argument.
- **CS0234 / CS0246 in MiniGames** ‚Äî `PolishMiniGame`, `CleanseMiniGame`, and `MiniGameBase` use `MemoryOrbInteractable` (in `HearthboundHollow.Player`). The MiniGames asmdef was missing the Player reference. Added.
- **CS0234 (preemptive) in Dialogue** ‚Äî When the user installs Yarn Spinner, `YarnVillageStateBridge.SaveAutopoint()` activates and references `HearthboundHollow.Save.SaveService`. The Dialogue asmdef now references Save proactively so no rework is needed at Yarn-install time.
- **"Project has invalid dependencies"** ‚Äî Unity 6 folded `com.unity.textmeshpro` into `com.unity.ugui` 2.0; the standalone package no longer resolves. Removed from `Packages/manifest.json`. The `Unity.TextMeshPro` assembly name is still valid (asmdefs that reference it continue to compile).

### Added ‚Äî Editor tooling (Phase 11)
- **`Editor/SeedAssetGenerator.cs`** ‚Äî Menu item `Hearthbound ‚Üí Create Mission 1-2 Seed Assets` creates all 17 canonical ScriptableObject assets at the expected paths:
  - `VillageState.asset`
  - `Mission01_OpeningTheHollow.asset`, `Mission02_TheWidowersRequest.asset`
  - `Doris.asset`, `Gerrold.asset`, `SilentLane.asset`
  - `Lavender.asset`, `Valerian.asset`
  - `DOR-001_FirstLoaves.asset`, `GER-007_SeventhMorning.asset`
  - `ECHO_DOR001_GER007.asset`
  - `MemoryMap_Doris.asset`, `MemoryMap_Gerrold.asset`
  - `Tariff_Erase.asset`, `Tariff_Cleanse.asset`, `Tariff_Listen.asset`, `Tariff_Defer.asset`
  - Saves ~30 minutes of right-click ‚Üí Create per asset.
  - Idempotent: skips assets that already exist; safe to re-run.
- **`Editor/HearthboundHollow.Editor.asmdef`** ‚Äî Editor-only assembly so menu code does not ship in runtime builds.
- **Menu item `Hearthbound ‚Üí Validate Mission 1-2 Seed Assets`** ‚Äî Logs which seed assets are missing.

### Added ‚Äî Runtime (Phase 11)
- **`Cutscene/ListenSceneSequencer.cs`** ‚Äî Mission 2 Listen-path Timeline driver. Plays the 3-min Listen cutscene, then routes to Memory Dream 2 Variant D. Cinemachine priority is set via reflection so the Cutscene asmdef does not take a hard `com.unity.cinemachine` compile dep (D-009).
- **`Settings/HearthboundInput.inputactions`** ‚Äî Pre-configured Input Actions asset:
  - **Move** (Vector2): WASD, Arrow keys, Gamepad left stick
  - **Interact** (Button): E key, Gamepad south button
  - **PointerPosition** (Vector2): mouse / touch position
  - **PointerActive** (Button): left mouse / touch press
  - **Pause** (Button): Escape, Gamepad start
  - Three control schemes: Keyboard&Mouse, Gamepad, Touch (mobile-ready).

### Added ‚Äî Tests (Phase 11)
- **`Tests/EditMode/SaveAndRippleTests.cs`** ‚Äî +13 NUnit tests, bringing EditMode coverage from 8 to 21:
  - `SaveServiceRoundTripTests` (5): round-trip, autosave slot, missing slot, empty label, delete slot
  - `RippleEngineTests` (4): Erase tariff, Listen tariff, coin clamp, integrity clamp
  - `MemoryNodeTests` (2): EffectiveTint override + tone fallback
  - `VillagerMemoryRuntimeTests` (2): Reveal + idempotency
- EditMode tests asmdef updated to reference `HearthboundHollow.Mission` (for RippleEngine).

### Decisions adopted in this release
- **D-008** Drop `com.unity.textmeshpro` from manifest ‚Äî Unity 6 folded TMP into `com.unity.ugui` 2.0.
- **D-009** ListenSceneSequencer integrates Cinemachine via reflection ‚Äî Cutscene asmdef stays Cinemachine-agnostic.
- **D-010** Seed assets are generated by Editor menu, not committed as .asset files ‚Äî avoids GUID/.meta race conditions; idempotent + safe to re-run.

### Files net delta (vs v0.1.0)
- **+4 C# files**: ListenSceneSequencer, SeedAssetGenerator, SaveAndRippleTests (+CoreTests still present).
- **+1 asmdef**: HearthboundHollow.Editor.
- **+1 Input Actions asset**: HearthboundInput.inputactions.
- **Modified**: SaveService.cs, manifest.json, MiniGames asmdef, Dialogue asmdef, EditMode tests asmdef, PROGRESS.md, CHANGELOG.md.
