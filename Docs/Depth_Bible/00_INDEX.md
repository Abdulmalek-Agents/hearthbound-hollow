## 0.1 Why This Bible Exists

The original Expert Panel Review gave Hearthbound Hollow a **6.0/10 overall success probability** with a **brilliant concept · catastrophic execution risk** verdict. The single largest existential threat was the writing burden (360–600 hand-authored vignettes), followed by art-budget gaps for Memory Dreams, mini-game fatigue, progression-goal voids, and the cozy/heavy-themes positioning collision.

This Depth Bible **does not** dilute the original vision. It **fortifies it** by:

1. **Multiplying depth without multiplying line-count.** Procedural systems, parameterized templates, and modular vignettes turn 600 hand-written memories into 60 hand-written *foundations* with 540 procedurally-recombined facets — at the same perceived narrative density.
2. **Adding 14 new mechanical pillars** (Memory Composting, Memory Bees, Letter Birds, Echo Holograms, Confession Booth, Memory Sommelier mode, etc.) that each address a specific review risk while creating fresh USPs.
3. **Engineering humor.** The original pitch reads beautifully but tonally somber. Humor is now a first-class system (Codex 07) — slapstick memory mishaps, the sarcastic Hollow's cat *Pickle*, satirical traveler arcs, the Memory Stock Exchange villain — all designed to make the cozy crowd laugh without breaking emotional weight.
4. **Hard-spec replayability.** Procedural memory generation, a weekly memory market, asynchronous co-op letter-birds, and a New-Game+ "Drifter" mode push lifetime hours from a projected 25 → **80+**.
5. **Codifying the team.** 15 specialist documents — each a self-contained, citable internal reference — form the studio's onboarding bible for every future writer, artist, engineer, and producer.

## 0.7 Where This Bible Disagrees With Itself

A healthy team argues. The following tensions are **preserved on purpose** — they are decisions for the producer to make at greenlight, not for the bible to pre-empt:

1. **Inara Vellis (Narrative) vs. Sven Aleko (Dreams)** — Inara argues 14–16 villagers, Sven argues 10 villagers + denser dreams. *Producer's call.*
2. **Mochi Tannenbaum (Humor) vs. Pell Doyne (Comfort)** — Mochi wants slapstick memory mishaps to be *unmissable*; Pell wants every comedic beat opt-out for the most fragile players. *Designed as togglable.*
3. **Ari Sukenobu (Roguelite) vs. Esme Cordray (Consequence)** — Ari wants procedural memory variety; Esme worries proceduralism dilutes the moral weight of choices. *Resolved via the Hand-Sealed/Procedural split.*
4. **Daud Reyes (Community) vs. Halvor Krieg (Risk)** — Daud wants Letter-Birds to be mandatory; Halvor wants them optional + offline-first. *Resolved as opt-in.*

These tensions are documented in their respective codices and revisited in `16 § 9 — Greenlight Decision Gates`.
